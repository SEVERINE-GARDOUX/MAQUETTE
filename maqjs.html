Le début en JavaScript

Tout d'abord il faut lié sa page javascript (js) a sa page HTML. La page js se lie a la fin car elle est lue en dernier. On la lie en inscrivant la balise script scr, entre les balises fermantes de body et HTML.

Une fois la page js créer et liée, le travail peut commencer.

Les variables

Qu'est ce qu'une variable ?

C'est une boîte où on stock des choses.
Exemple : let (variable) unText (nom de la varible) = "Voici un texte"(contenu de la varible)
On déclare une variable avec let ou const.

Les variables let et const sont différentes. En effet, let est une instruction tandis que const est une constante.
Cependant, toutes deux déclarent une variable qui fonctionne que dans le bloc dans lequel celle-ci est déclarée. Pour qu'une variable soit fonctionnelle, nous devons la déclarer et l'éxécuter (l'appeler)
