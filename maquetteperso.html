Le titre SEO

Il est très important pour le référencement, il faut bien le choisir !

Ce titre se trouve dans l'onglet de la page internet.
Vous pouvez insérer une image a coté. Pour cela, il suffit de mettre en dessous le titre seo :
link rel="shortcut icon" href="fichier de l'image"
Vous aurez ainsi une image a côté du titre de votre page
